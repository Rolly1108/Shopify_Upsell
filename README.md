# Shopify_Upsell

## Overview
Shopify Upsell is a Shopify app designed to help merchants increase Average Order Value (AOV) and improve conversion through upsell, cross-sell, and product recommendation experiences across the buyer journey.
This module is a core part of the Shopify Free Gift Upsell Bundles app, providing upsell functionality on the Product page, Cart page, Post-purchase page, Thank you page, and Checkout page.

## Objectives
- Help merchants create upsell offers quickly
- Improve AOV and conversion rate
- Provide simple setup for non-technical merchants
- Offer actionable analytics for performance optimization

## Main modules
- Dashboard
- Offer Management
- Offer Builder
- Analytics
- Settings

## Target users
- Shopify merchants
- Small and medium businesses
- Growing brands that want to increase revenue per order
