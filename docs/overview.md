# Project Overview

## Product name
Shopify Upsell

## Product summary
Shopify Upsell is a Shopify app that enables merchants to create upsell and cross-sell offers at different stages of the shopping journey, such as product page, cart, and post-purchase.

## Business goal
Increase merchant revenue by improving:
- Average Order Value (AOV)
- Conversion rate
- Upsell offer adoption

## Problem statement
Many merchants want to increase order value but:
- do not know where to place upsell offers effectively
- do not have time to set up complicated logic
- cannot clearly measure upsell performance
- need a simple tool that works without heavy customization

## Target users
### Primary users
- Shopify merchants
- Store owners
- Marketing managers
- E-commerce operators

### Secondary users
- Agencies managing multiple Shopify stores
- Store staff responsible for campaign setup

## User needs
- Easy setup
- Clear offer management
- Flexible targeting rules
- Simple analytics
- Fast activation

## MVP scope
- Merchant onboarding
- Dashboard
- Offer list
- Create/Edit upsell offer
- Basic targeting conditions
- AI-generated offer recommendations
- Product page upsell
- Cart upsell
- Post purchase page upsell
- Thankyou page upsell
- Checkout page upsell
- Basic analytics
- App settings

## Out of scope for MVP
- Advanced customer segmentation
- A/B testing
- Multi-language content management
- Deep revenue attribution model
- Enterprise permission system

## Product principles
- Easy for non-technical merchants
- Fast to launch first offer
- Clear performance visibility
- Minimal setup friction
- Scalable for future advanced features

## Assumptions
- Most merchants want quick wins rather than complex setup
- Merchants need both performance visibility and guided setup
- A clean and simple admin experience increases feature adoption

## Success criteria
- Merchant can create and activate first offer without support
- Merchant understands core metrics from dashboard
- Merchant can manage offers confidently
- Merchant sees value within the first few sessions
