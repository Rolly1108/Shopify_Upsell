# User Flow

## 1. Merchant onboarding flow
1. Merchant installs the app
2. Merchant lands on welcome screen
3. Merchant sees value proposition and setup steps
4. Merchant goes to dashboard or starts first offer setup
5. Merchant creates first offer
6. Merchant activates the offer
7. Merchant returns to dashboard to monitor performance

## 2. Merchant offer creation flow
1. Merchant opens Offer List
2. Merchant clicks "Create Offer"
3. Merchant selects page upsell: Product page/ Cart page/ Post purchase page/ Thankyou page/ Checkout page
4. Merchant enters offer name
5. Merchant configures trigger and conditions
6. Merchant configures offer defines recommended product and discount
7. Merchant reviews configuration
8. Merchant saves activates
9. Merchant sees success confirmation

## 3. Merchant offer management flow
1. Merchant opens Offer List
2. Merchant searches or filters offers
3. Merchant opens offer detail
4. Merchant edits or duplicates offer
5. Merchant enables/disables the offer
6. Merchant returns to list

## 4. Merchant analytics flow
1. Merchant opens Dashboard or Analytics page
2. Merchant reviews KPI summary
3. Merchant checks top performing offers
4. Merchant identifies weak or strong performers
5. Merchant goes to edit an offer based on performance insight

## 5. Buyer product page upsell flow
1. Buyer lands on product page
2. Buyer reviews primary product
3. Buyer sees upsell recommendation
4. Buyer adds recommended item
5. Buyer proceeds to cart

## 6. Buyer cart upsell flow
1. Buyer opens cart
2. Buyer sees upsell suggestion
3. Buyer adds suggested item
4. Buyer continues to checkout

## 7. Buyer post purchase page upsell flow
One click upsell
1. Buyer completes the initial purchase.
2. Upsell products are displayed on the post-purchase page.
3. Buyer clicks [Pay now].
4. The system reuses the existing payment and buyer information to complete the transaction.

## 8. Buyer thankyou page upsell flow
1. Buyer completes the initial purchase.
2. Upsell products are displayed on the thankyou page.
3. Buyer adds suggested item
4. Buyer continues to checkout

## 9. Buyer checkout page upsell flow
1. Buyer click checkout.
2. Upsell products are displayed on the checkout page.
3. Buyer adds suggested item
4. Buyer continues to pay now

## 10. New merchant success path
1. Install app
2. Understand app value quickly
3. Create first offer with low friction
4. Activate first offer
5. See initial analytics
6. Create more offers

## 11. Key UX goals across flows
- Reduce setup friction
- Make next actions obvious
- Keep screens simple and guided
- Show value as early as possible
